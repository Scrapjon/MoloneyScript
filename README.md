# MoloneyScript
Compiled programming language

Currently being implemented in Python as a proof of concept prototype. Will be remade in another language (most likely  C# or C++) once a functioning version has been made in Python.

## What is the goal of MoloneyScript?

I love metaprogramming, people I work on projects hate me for it sometimes but that is just my nature. Originally this was designed to be a preprocessor on steroids, taking inspiration from Unreal Engine's header tool (UHT) but I quickly realised that I can build off of the C and C++ language in the same way that C built off of assembly. Hence this is what my project aims to be.


This project is currently in the stages of having the language specifications planned out.
